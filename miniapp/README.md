# 小程序端

当前先实现一个 **小程序风格的 H5 原型页面**，由后端直接托管，便于你快速联调和演示。

## 已完成页面
- 首页
- 借用页面
- 申领页面
- 我的记录页面

## 已实现能力
- 切换模拟当前用户
- 查看设备/耗材列表
- 提交借用申请
- 提交耗材申领
- 查看当前用户自己的借用/申领记录

## 打开方式
1. 在 backend 目录执行 `npm start`
2. 浏览器访问 `http://localhost:3000/miniapp/`
